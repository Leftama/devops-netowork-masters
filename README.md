## Ejecutar localmente

**Prerrequisitos:** Node.js

1. Instalar dependencias:
   `npm install`

2. Configurar `GEMINI_API_KEY` en [.env.local](.env.local) con tu clave API de Gemini

3. Ejecutar la aplicación:
   `npm run dev`
